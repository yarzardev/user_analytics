# How to Run

1. Create a virtual environment:
	```bash
	python -m venv venv
	```
2. Activate the virtual environment:
	```bash
	source venv/bin/activate
	```
3. Install dependencies:
	```bash
	pip install -r requirements.txt
	```
4. Run the application:
	```bash
	python main.py
	```
